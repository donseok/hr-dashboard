# DK HR 통합관리 - 퀵 매뉴얼

**빠른 참조 가이드** | 최종 수정일: 2026-03-04

---

## 시스템 접속

| 항목 | 내용 |
|------|------|
| 접속 방법 | 웹 브라우저에서 시스템 URL 접속 |
| 로그인 | 이메일 + 비밀번호 또는 SSO 로그인 |
| 로그아웃 | 우측 상단 아바타 > "로그아웃" 클릭 |

---

## 네비게이션 모듈

| 메뉴 | 경로 | 주요 기능 |
|------|------|-----------|
| 개요 | `/` | HR 핵심 KPI 종합 대시보드, 인력 변동/이직률 차트 |
| 채용 | `/recruitment` | 채용 파이프라인, 전환율, 소요기간, 오퍼 수락률 |
| 인력 | `/workforce` | 부서별 인원, 인력 구성, 입퇴사 추이 |
| 성과 | `/performance` | 성과 평가 현황, 목표 달성률, 등급 분포 |
| 문화 | `/culture` | eNPS, 직원 만족도, 조직문화 지표 |
| 육성 | `/development` | 교육 프로그램 현황, 수료율, 역량 개발 |
| DEI | `/dei` | 다양성 지수, 형평성, 포용성 지표 |
| 라이프사이클 | `/lifecycle` | 직원 여정 단계, 리텐션 분석 |
| 리포트 | `/reports` | 맞춤형 리포트 생성, 데이터 다운로드 |
| 설정 | `/settings` | 일반, 알림, 표시 설정 관리 |

---

## 화면 구성

| 영역 | 위치 | 기능 |
|------|------|------|
| 사이드바 | 좌측 | 모듈 네비게이션 (접기/펼치기 가능) |
| 헤더 | 상단 | 글로벌 검색, 테마 토글, 알림, 사용자 메뉴 |
| 대시보드 | 중앙 | KPI 카드, 차트, 필터, 데이터 테이블 |

---

## 키보드 단축키

| 단축키 | 기능 |
|--------|------|
| `Cmd+K` (Mac) / `Ctrl+K` (Windows) | 글로벌 검색 열기 |
| `Esc` | 검색/팝업 닫기 |

---

## 필터 사용법

| 동작 | 방법 |
|------|------|
| 필터 패널 열기 | "필터" 버튼 클릭 |
| 날짜 범위 선택 | 날짜 선택기에서 시작일/종료일 지정 |
| 부서 필터링 | 부서 드롭다운에서 부서 선택 (복수 선택 가능) |
| 개별 필터 제거 | 필터 칩의 X 버튼 클릭 |
| 전체 필터 초기화 | "초기화" 버튼 클릭 |
| 필터 패널 접기 | 필터 패널 우측 상단 X 버튼 클릭 |

---

## KPI 카드 읽는 법

| 요소 | 설명 |
|------|------|
| 시그널 표시등 | 초록(양호), 노랑(주의), 빨강(위험) |
| 큰 숫자 | 현재 지표 값 |
| 화살표 + 퍼센트 | 전기 대비 변동률 |
| 미니 차트 (스파크라인) | 최근 추세 시각화 |
| 물음표 아이콘 | 마우스 올리면 지표 설명 표시 |

---

## 풍선도움말 (Tooltip)

| 위치 | 사용 방법 |
|------|-----------|
| KPI 카드 | 제목 옆 물음표(info) 아이콘에 마우스 올리기 |
| 차트 제목 | 제목 옆 물음표(info) 아이콘에 마우스 올리기 |
| 차트 데이터 포인트 | 차트 위 마우스 올리면 상세 수치 표시 |
| 사이드바 (축소 시) | 아이콘에 마우스 올리면 메뉴 이름 표시 |

## 테마 변경

| 모드 | 설명 | 전환 방법 |
|------|------|-----------|
| 라이트 | 밝은 배경 (기본) | 헤더의 테마 토글 버튼 클릭 |
| 다크 | 어두운 배경 | 토글 버튼 재클릭 |
| 시스템 | OS 설정 연동 | 토글 버튼 재클릭 |

설정 페이지에서도 변경 가능: 설정 > 표시 탭 > 테마 선택

---

## 리포트

| 동작 | 방법 |
|------|------|
| 리포트 목록 보기 | 사이드바 > "리포트" 클릭 |
| 새 리포트 생성 | "새 리포트 생성" 버튼 클릭 |
| 리포트 다운로드 | 상태가 "완료"인 리포트의 "다운로드" 버튼 클릭 |

---

## 설정 탭

| 탭 | 설정 항목 |
|----|-----------|
| 일반 | 기본 기간 (30일/90일/6개월/1년), 데이터 새로고침 간격 (실시간/1분/5분/15분) |
| 알림 | 알림 수신 조건, 알림 채널 설정 |
| 표시 | 테마 모드, 대시보드 표시 방식 |

---

## 알림

| 동작 | 방법 |
|------|------|
| 알림 확인 | 헤더 우측 벨 아이콘 클릭 |
| 읽지 않은 알림 | 벨 아이콘 우측 상단 빨간 뱃지로 표시 |
| 모두 읽음 처리 | 알림 패널 상단 "모두 읽음" 클릭 |

---

## 사이드바 접기/펼치기

| 동작 | 방법 |
|------|------|
| 접기 | 사이드바 하단 `<<` 버튼 클릭 |
| 펼치기 | 축소된 사이드바 하단 `>>` 버튼 클릭 |
| 축소 시 메뉴 확인 | 아이콘에 마우스 올리면 메뉴 이름 표시 |
