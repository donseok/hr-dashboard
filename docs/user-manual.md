## 1. 소개

### 1.1 시스템 개요

DK HR 통합관리 시스템은 조직의 인사 데이터를 하나의 대시보드에서 통합 관리하고 분석할 수 있는 웹 기반 HR 애널리틱스 플랫폼입니다.

### 1.2 주요 기능

| 기능 | 설명 |
|------|------|
| 종합 대시보드 | 주요 HR 지표를 KPI 카드와 차트로 한눈에 파악 |
| 7개 전문 모듈 | 채용, 인력, 성과, 문화, 육성, DEI, 라이프사이클 분석 |
| 실시간 데이터 | 실시간 데이터 연동으로 최신 현황 반영 |
| 맞춤형 리포트 | 리포트 생성 및 데이터 내보내기 |
| 다크 모드 | 라이트/다크/시스템 테마 지원 |
| 풍선도움말 | KPI 및 차트에 대한 상세 설명 제공 |

### 1.3 지원 브라우저

| 브라우저 | 최소 버전 |
|----------|-----------|
| Chrome | 90 이상 |
| Firefox | 90 이상 |
| Safari | 15 이상 |
| Edge | 90 이상 |

## 2. 로그인

### 2.1 접속 방법

1. 웹 브라우저에서 시스템 URL에 접속합니다.
2. 로그인 화면이 표시됩니다.

### 2.2 인증 절차

| 단계 | 행동 |
|------|------|
| 1 | 이메일 주소를 입력합니다 |
| 2 | 비밀번호를 입력합니다 |
| 3 | "로그인" 버튼을 클릭합니다 |
| 4 | SSO 사용 시, "SSO 로그인" 버튼을 클릭하여 사내 인증 시스템으로 이동합니다 |

### 2.3 로그아웃

1. 우측 상단의 프로필 아바타를 클릭합니다.
2. 드롭다운 메뉴에서 "로그아웃"을 선택합니다.

## 3. 화면 구성

DK HR 통합관리 시스템의 화면은 크게 세 영역으로 구성됩니다.

### 3.1 전체 레이아웃

| 영역 | 위치 | 설명 |
|------|------|------|
| 사이드바 | 좌측 | 네비게이션 메뉴, 모듈 목록 |
| 헤더 | 상단 | 검색, 알림, 사용자 메뉴 |
| 대시보드 영역 | 중앙 | 선택한 모듈의 콘텐츠 표시 |

### 3.2 사이드바 (네비게이션)

사이드바는 시스템의 모든 모듈에 접근할 수 있는 네비게이션 메뉴입니다.

| 메뉴 항목 | 아이콘 | 설명 |
|-----------|--------|------|
| 개요 | 대시보드 아이콘 | HR 종합 개요 대시보드 |
| 채용 | UserPlus | 채용 파이프라인 및 분석 |
| 인력 | Users | 인력 현황 및 분석 |
| 성과 | TrendingUp | 성과 관리 및 평가 |
| 문화 | Heart | 조직 문화 및 만족도 |
| 육성 | GraduationCap | 교육 및 역량 개발 |
| DEI | Globe | 다양성, 형평성, 포용성 |
| 라이프사이클 | RefreshCw | 직원 여정 관리 |
| 리포트 | Document | 맞춤형 리포트 생성 |
| 설정 | Gear | 시스템 설정 |

**사이드바 접기/펼치기:**
- 사이드바 하단의 접기 버튼(<<)을 클릭하면 아이콘만 표시되는 축소 모드로 전환됩니다.
- 축소 모드에서 아이콘에 마우스를 올리면 메뉴 이름이 툴팁으로 표시됩니다.
- 접기 버튼을 다시 클릭하면 전체 사이드바가 펼쳐집니다.

### 3.3 헤더

헤더 영역에는 다음 기능이 제공됩니다.

| 기능 | 위치 | 설명 |
|------|------|------|
| 글로벌 검색 | 좌측 | 시스템 전체에서 데이터를 검색 (단축키: Cmd+K 또는 Ctrl+K) |
| 테마 토글 | 중앙~우측 | 라이트/다크 모드 전환 버튼 |
| 알림 벨 | 우측 | 새 알림 확인, 읽지 않은 알림 수 표시 |
| 사용자 아바타 | 최우측 | 프로필, 설정, 로그아웃 메뉴 |

**알림 기능 상세:**
1. 알림 벨 아이콘을 클릭하면 알림 패널이 열립니다.
2. 읽지 않은 알림이 있으면 벨 아이콘 우측 상단에 빨간 뱃지로 개수가 표시됩니다.
3. "모두 읽음" 버튼을 클릭하면 모든 알림을 읽음 처리합니다.

### 3.4 대시보드 영역

사이드바에서 선택한 모듈의 콘텐츠가 표시되는 주요 영역입니다.

- 상단: 페이지 제목 및 설명
- 필터 영역: 날짜, 부서 등 필터 선택
- KPI 카드 그리드: 핵심 지표 요약
- 차트 영역: 시각화 그래프

## 4. 개요 대시보드

개요 대시보드는 시스템의 첫 화면으로, 조직의 HR 핵심 지표를 종합적으로 보여줍니다.

### 4.1 KPI 카드

KPI 카드는 핵심 인사 지표를 요약하여 보여주는 카드 형태의 위젯입니다.

| KPI | 설명 | 단위 |
|-----|------|------|
| 총 인원 | 현재 재직 중인 전체 직원 수 | 명 |
| 이직률 | 일정 기간 내 퇴사한 직원의 비율 | % |
| 채용 소요일 | 채용 공고부터 입사까지 평균 소요 기간 | 일 |
| 직원 만족도 | 직원 만족도 설문 평균 점수 | /5.0 |

**KPI 카드 구성 요소:**

| 요소 | 설명 |
|------|------|
| 시그널 표시등 | 지표 상태를 색상으로 표시 (초록: 양호, 노랑: 주의, 빨강: 위험) |
| 지표명 | KPI 항목 이름 |
| 현재 값 | 현재 지표 수치 |
| 변동률 | 전기 대비 증감률 (%, 화살표로 방향 표시) |
| 스파크라인 | 추세를 보여주는 미니 차트 |
| 도움말 아이콘 | 클릭 시 해당 KPI에 대한 상세 설명 표시 |

### 4.2 차트

개요 대시보드에는 다음 차트가 제공됩니다.

| 차트명 | 유형 | 설명 |
|--------|------|------|
| 인력 변동 추이 | 스택 바 차트 | 월별 입사/퇴사 현황 비교 |
| 이직률 트렌드 | 라인 차트 | 분기별 이직률 vs 업계 평균 비교 |

**차트 사용법:**
1. 차트 위에 마우스를 올리면 해당 데이터 포인트의 상세 수치가 툴팁으로 표시됩니다.
2. 차트 하단의 범례(Legend)를 클릭하면 특정 데이터 시리즈를 표시/숨기기 할 수 있습니다.

## 5. 모듈별 사용법

### 5.1 인력현황 (인력)

부서별 인원 현황과 인력 구성을 분석하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 부서별 인원 | 각 부서의 현재 인원 수 및 분포 확인 |
| 입퇴사 추이 | 월별/분기별 입퇴사 현황 추세 분석 |
| 인력 구성 분석 | 직급, 연령, 근속 기간별 인력 구조 파악 |

**주요 KPI:** 총 인원, 이직률, 평균 근속 기간

**사용 절차:**
1. 사이드바에서 "인력"을 클릭합니다.
2. KPI 카드에서 핵심 인력 지표를 확인합니다.
3. 차트를 통해 인력 변동 추이를 분석합니다.
4. 필터를 사용하여 특정 부서나 기간의 데이터를 조회합니다.

### 5.2 채용

채용 파이프라인과 채용 효율성을 분석하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 채용 파이프라인 | 지원 > 서류심사 > 면접 > 오퍼 > 입사 단계별 현황 |
| 전환율 | 각 채용 단계 간 전환 비율 분석 |
| 채용 소요기간 | 공고 게시부터 최종 합격까지 소요 기간 추적 |
| 오퍼 수락률 | 오퍼 대비 수락 비율 |

**주요 KPI:** 공석 수, 채용 전환율, 채용 소요일, 오퍼 수락률

**사용 절차:**
1. 사이드바에서 "채용"을 클릭합니다.
2. 퍼널 차트에서 채용 파이프라인의 각 단계별 전환율을 확인합니다.
3. KPI 카드에서 채용 효율 지표를 확인합니다.
4. 필터를 사용하여 특정 부서나 직무의 채용 현황을 조회합니다.

### 5.3 성과관리

직원 성과 평가 현황과 목표 달성률을 관리하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 평가 현황 | 성과 평가 진행 상태 및 완료율 확인 |
| 목표 달성률 | 개인/팀별 목표 달성 현황 분석 |
| 평가 분포 | 성과 등급별 직원 분포 확인 |

**주요 KPI:** 평가 완료율, 평균 목표 달성률, 성과 등급 분포

**사용 절차:**
1. 사이드바에서 "성과"를 클릭합니다.
2. KPI 카드에서 성과 관리 핵심 지표를 확인합니다.
3. 차트를 통해 성과 등급 분포와 목표 달성 현황을 분석합니다.

### 5.4 인재개발 (육성)

교육 프로그램 현황과 직원 역량 개발을 관리하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 교육 프로그램 | 진행 중인 교육 과정 목록 및 현황 |
| 수료율 | 교육 과정별 수료 현황 및 완료율 |
| 역량 개발 추이 | 직원 역량 개발 트렌드 분석 |

**주요 KPI:** 교육 수료율, 교육 참여율, 교육 ROI

**사용 절차:**
1. 사이드바에서 "육성"을 클릭합니다.
2. KPI 카드에서 교육 관련 핵심 지표를 확인합니다.
3. 차트를 통해 수료율 추이와 교육 효과를 분석합니다.

### 5.5 라이프사이클

직원 여정 단계별 현황과 리텐션을 분석하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 직원 여정 | 입사 > 온보딩 > 성장 > 이탈 방지 단계별 현황 |
| 리텐션 분석 | 근속 기간별 잔류율 분석 |
| 이탈 위험 예측 | 퇴사 위험 직원 조기 식별 |

**주요 KPI:** 리텐션율, 평균 근속 기간, 온보딩 완료율

**사용 절차:**
1. 사이드바에서 "라이프사이클"을 클릭합니다.
2. KPI 카드에서 직원 여정 핵심 지표를 확인합니다.
3. 차트를 통해 직원 여정 단계별 현황을 분석합니다.

### 5.6 조직문화 (문화)

조직문화 지표와 직원 만족도를 분석하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| eNPS | 직원 순추천 지수(Employee Net Promoter Score) 추적 |
| 만족도 추이 | 직원 만족도 설문 결과 변화 추세 |
| 문화 지표 | 조직문화 관련 종합 지표 분석 |

**주요 KPI:** eNPS, 직원 만족도, 참여율

**사용 절차:**
1. 사이드바에서 "문화"를 클릭합니다.
2. KPI 카드에서 조직문화 핵심 지표를 확인합니다.
3. 차트를 통해 만족도 추이와 eNPS 변동을 분석합니다.

### 5.7 DEI

다양성(Diversity), 형평성(Equity), 포용성(Inclusion) 지표를 분석하는 모듈입니다.

| 기능 | 설명 |
|------|------|
| 다양성 지수 | 조직 구성의 다양성을 나타내는 종합 지수 |
| 성별/연령 분포 | 성별, 연령대별 인력 구성 분석 |
| 형평성 지표 | 직급별, 부서별 형평성 분석 |
| 포용성 설문 | 포용성 관련 직원 설문 결과 분석 |

**주요 KPI:** 다양성 지수, 성별 비율, 형평성 점수

**사용 절차:**
1. 사이드바에서 "DEI"를 클릭합니다.
2. KPI 카드에서 DEI 핵심 지표를 확인합니다.
3. 차트를 통해 다양성 지수 및 구성 분포를 분석합니다.

## 6. 필터 사용법

필터 기능을 활용하면 원하는 조건으로 데이터를 좁혀 볼 수 있습니다.

### 6.1 필터 패널 열기

| 단계 | 행동 |
|------|------|
| 1 | 대시보드 상단의 "필터" 버튼을 클릭합니다 |
| 2 | 필터 패널이 펼쳐집니다 |
| 3 | 필터를 접으려면 우측 상단의 X 버튼을 클릭합니다 |

### 6.2 날짜 범위 선택

| 단계 | 행동 |
|------|------|
| 1 | 필터 패널에서 날짜 선택기를 클릭합니다 |
| 2 | 시작일과 종료일을 선택합니다 |
| 3 | 선택한 날짜 범위의 데이터가 자동으로 반영됩니다 |

### 6.3 부서 선택

| 단계 | 행동 |
|------|------|
| 1 | 필터 패널에서 부서 드롭다운을 클릭합니다 |
| 2 | 조회할 부서를 하나 이상 선택합니다 |
| 3 | 선택한 부서의 데이터만 필터링되어 표시됩니다 |

### 6.4 필터 칩(Chip)

적용된 필터는 필터 패널 아래에 칩(Chip) 형태로 표시됩니다.

| 동작 | 방법 |
|------|------|
| 개별 필터 제거 | 필터 칩의 X 버튼 클릭 |
| 전체 필터 초기화 | "초기화" 버튼 클릭 |

### 6.5 필터 조합

- 여러 필터를 동시에 적용할 수 있습니다.
- 날짜 범위와 부서를 함께 선택하면 교차 조건으로 데이터가 필터링됩니다.
- 필터 적용 후 모든 KPI 카드와 차트가 자동으로 갱신됩니다.

## 7. 풍선도움말 활용

DK HR 통합관리 시스템은 각 KPI 카드와 차트에 풍선도움말(Tooltip)을 제공하여 지표의 의미를 쉽게 이해할 수 있도록 돕습니다.

### 7.1 도움말 아이콘 (info 아이콘)

| 위치 | 아이콘 | 설명 |
|------|--------|------|
| KPI 카드 제목 옆 | 원형 물음표 아이콘 | 해당 KPI 지표에 대한 상세 설명 표시 |
| 차트 제목 옆 | 원형 물음표 아이콘 | 해당 차트에 대한 상세 설명 표시 |
| 사이드바 (축소 모드) | 자동 표시 | 메뉴 항목 이름 표시 |

### 7.2 사용 방법

1. KPI 카드 또는 차트 제목 옆의 info 아이콘(원형 물음표)에 마우스를 올립니다.
2. 해당 지표에 대한 설명이 풍선도움말로 표시됩니다.
3. 마우스를 아이콘에서 벗어나면 풍선도움말이 사라집니다.

### 7.3 제공되는 도움말 예시

| KPI | 도움말 내용 |
|-----|------------|
| 총 인원 | 현재 재직 중인 전체 직원 수입니다 |
| 이직률 | 일정 기간 내 퇴사한 직원의 비율입니다 |
| 평균 근속 기간 | 전체 직원의 평균 근속 기간입니다 |
| 공석 수 | 현재 채용이 진행 중인 공석 수입니다 |
| 채용 전환율 | 지원자 대비 최종 채용된 인원의 비율입니다 |
| 채용 소요일 | 채용 공고 게시부터 입사 확정까지 소요되는 평균 일수입니다 |
| eNPS | 직원 순추천 지수로, 조직 추천 의향을 측정합니다 |
| 직원 만족도 | 직원 만족도 설문 결과의 평균 점수입니다 |
| 다양성 지수 | 조직 구성의 다양성을 나타내는 종합 지수입니다 |

## 8. 리포트

### 8.1 리포트 목록 확인

1. 사이드바에서 "리포트"를 클릭합니다.
2. 생성된 리포트 목록이 테이블 형태로 표시됩니다.

| 열 | 설명 |
|----|------|
| 리포트명 | 리포트 제목 |
| 유형 | 정기 또는 수시 |
| 최종 생성일 | 마지막으로 생성된 날짜 |
| 상태 | 완료 또는 생성 중 |
| 액션 | 다운로드 버튼 |

### 8.2 리포트 유형

| 유형 | 설명 |
|------|------|
| 정기 리포트 | 월간/분기별 등 주기적으로 자동 생성되는 리포트 |
| 수시 리포트 | 필요에 따라 수동으로 생성하는 리포트 |

### 8.3 새 리포트 생성

| 단계 | 행동 |
|------|------|
| 1 | 리포트 페이지 우측 상단의 "새 리포트 생성" 버튼을 클릭합니다 |
| 2 | 리포트 유형과 포함할 데이터 범위를 선택합니다 |
| 3 | 리포트 이름을 입력하고 생성을 확인합니다 |
| 4 | 생성이 완료되면 상태가 "완료"로 변경됩니다 |

### 8.4 리포트 다운로드

- 상태가 "완료"인 리포트의 "다운로드" 버튼을 클릭합니다.
- "생성 중" 상태의 리포트는 다운로드 버튼이 비활성화됩니다.

### 8.5 제공되는 기본 리포트

| 리포트명 | 유형 | 설명 |
|----------|------|------|
| 월간 인력 현황 리포트 | 정기 | 월별 인력 변동 및 현황 종합 보고서 |
| 분기별 이직 분석 | 정기 | 분기별 이직률 및 원인 분석 |
| 채용 효율 분석 | 수시 | 채용 파이프라인 효율성 분석 |
| DEI 현황 리포트 | 정기 | 다양성, 형평성, 포용성 지표 보고서 |
| 교육 ROI 분석 | 수시 | 교육 투자 대비 효과 분석 |

## 9. 테마 변경

DK HR 통합관리 시스템은 라이트 모드, 다크 모드, 시스템 설정 연동의 세 가지 테마를 지원합니다.

### 9.1 테마 모드

| 모드 | 설명 |
|------|------|
| 라이트 | 밝은 배경에 어두운 텍스트 (기본값) |
| 다크 | 어두운 배경에 밝은 텍스트 |
| 시스템 | 운영체제의 다크 모드 설정을 자동으로 따름 |

### 9.2 헤더에서 테마 전환

| 단계 | 행동 |
|------|------|
| 1 | 헤더 우측의 테마 토글 버튼(해/달 아이콘)을 클릭합니다 |
| 2 | 클릭할 때마다 라이트 > 다크 > 시스템 순으로 전환됩니다 |
| 3 | 선택한 테마가 즉시 적용됩니다 |

### 9.3 설정 페이지에서 테마 변경

1. 사이드바에서 "설정"을 클릭합니다.
2. "표시" 탭을 선택합니다.
3. 테마 설정에서 원하는 모드를 선택합니다.

## 10. 설정

### 10.1 설정 접근 방법

사이드바 하단의 "설정"을 클릭하거나, 사용자 아바타 메뉴에서 "설정"을 선택합니다.

### 10.2 일반 설정

| 설정 항목 | 설명 | 선택 옵션 |
|-----------|------|-----------|
| 기본 기간 | 대시보드의 기본 데이터 조회 기간 | 최근 30일, 최근 90일, 최근 6개월, 최근 1년 |
| 데이터 새로고침 간격 | 데이터 자동 갱신 주기 | 실시간, 1분, 5분, 15분 |

### 10.3 알림 설정

| 설정 항목 | 설명 |
|-----------|------|
| 알림 수신 조건 | 어떤 이벤트에 대해 알림을 받을지 설정 |
| 알림 채널 | 시스템 알림, 이메일 등 수신 채널 선택 |

### 10.4 표시 설정

| 설정 항목 | 설명 |
|-----------|------|
| 테마 모드 | 라이트/다크/시스템 모드 선택 |
| 대시보드 표시 방식 | 차트 및 카드 레이아웃 설정 |

### 10.5 설정 저장

- 설정을 변경한 후 "저장" 버튼을 클릭하여 변경사항을 반영합니다.
- 저장하지 않고 페이지를 이동하면 변경사항이 유지되지 않습니다.
